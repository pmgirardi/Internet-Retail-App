## Installation

`npm init`

`npm install mysql2`

`npm install sequelize`

`npm install dotenv`
  
## Usage

Run the following commands in CLI in directory of project:

`mysql -u root -p`

`source db/schema.sql`

`quit`

`npm run seed`
  
`npm start`
